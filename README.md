Personal atomic image built with BlueBuild (https://github.com/blue-build/) and based on the Universal Blue main image (https://github.com/ublue-os/main). Uses the niri compositor (https://github.com/niri-wm/niri). Work in progress.

I don't recommend using this without my dotfiles but it may be helpful to read my recipe.yml to help with your own custom image. Use at your own risk.
